Работы
=====================
Данный репозиторий представляет собой сбор интересных работ на HTML, JS, CSS. По мере изучения JS, в некоторых работах стал применяться jQuery.

SLIDER
=====================
Представляет из себя полностью рабочий слайдер.
При помощи JS осуществляется добавление всех изображений на страницу и непосредственно в сам слайдер.

Адаптивная страница
=====================
Пробное написание адаптивной страницы в шуточной форме.

Игра
=====================
Пробное написание скрипта на взаимодействие пользователя с объект на странице.


Печаталка
=====================
Данная работа в основном написанна на JS.
При нажатие на любую клавишу на клавиатуре с символами в случайном месте видимой части страници появляется данный символ со случайным цветом и размером.

Часы
=====================
Не большой скрипт написанный на JS.
Показывают время в часах, минутах и секундах.
